We welcome contributions from the community! If you participated in CTF Bootcamp and would like to share your solution:

### Contribution Guidelines

1. **Fork this repository**
2. **Create a new branch** for your writeup
   ```bash
   git checkout -b writeup/category-challenge-name
   ```
3. **Follow the writeup template** provided above
4. **Add your writeup** to the appropriate category folder
5. **Include any necessary files** in the `files/` subdirectory
6. **Update the main README.md** to link your writeup
7. **Submit a Pull Request** with a clear description
