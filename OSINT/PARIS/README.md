Summary

This challenge, PARIS, is an OSINT (open-source intelligence) exercise from the Bootcamp Linadmo / CyberDune CTF. You're given one or more pieces of challenge material (image(s), text snippet, metadata, or a short webpage). Using publicly available tools and investigation techniques, you must find the physical location referenced by the challenge and extract the flag.

This README.md describes a reproducible methodology, tools, and an example workflow to solve the challenge in a forensics- and privacy-respecting CTF environment.
